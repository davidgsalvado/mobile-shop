# Component Reference

## Layout Components

### `Layout` (`src/components/layout/Layout.tsx`)
Root layout wrapper. Renders the `Header` and an `<Outlet />` for nested routes.

### `Header` (`src/components/layout/Header.tsx`)
Application header with three sections:
- **Left**: App title ("PhoneHub") — links to home (`/`).
- **Center/Below**: Breadcrumbs showing the current navigation path (Home → Product Name).
- **Right**: Shopping cart icon with badge showing item count from `cartStore`.

## Page Components

### `ProductListPage` (`src/pages/ProductListPage.tsx`)
- Fetches all products via `useProducts()`.
- Renders a search input that filters by brand and model in real-time.
- Displays a count of filtered results.
- Renders `ProductGrid` with filtered products.

### `ProductDetailPage` (`src/pages/ProductDetailPage.tsx`)
- Fetches product detail via `useProductDetail(id)`.
- Two-column layout: product image (left) + details & actions (right).
- Displays product specs via `useProductSpecs()` hook.
- Storage and color selectors with auto-select when only one option.
- "Add to Cart" button triggers `useAddToCart()` mutation.
- Shows loading skeleton while fetching.
- Updates breadcrumb title via `detailStore`.

### `NotFoundPage` (`src/pages/NotFoundPage.tsx`)
Simple 404 fallback page.

## Product Components

### `ProductCard` (`src/components/product/ProductCard.tsx`)
Card component for the product list. Shows:
- Product image (with hover zoom effect)
- Brand (uppercase, small text)
- Model name
- Price (or "Price unavailable" for invalid values)
- Optional "NEW" badge

Links to `/product/:id`.

### `ProductGrid` (`src/components/product/ProductGrid.tsx`)
Responsive grid layout:
- 1 column on mobile
- 2 columns on `sm` (≥640px)
- 3 columns on `lg` (≥1024px)
- 4 columns on `xl` (≥1280px)

Shows a skeleton loader during loading and an empty state message when no products match.

## Custom Hooks

### `useProducts` (`src/hooks/useProducts.ts`)
Wraps `useQuery` for fetching the product list. Query key: `['products']`.

### `useProductDetail` (`src/hooks/useProductDetail.ts`)
Wraps `useQuery` for fetching a single product. Query key: `['product', id]`. Disabled when `id` is falsy.

### `useAddToCart` (`src/hooks/useAddToCart.ts`)
Wraps `useMutation` for the cart API. On success, updates the cart count in `cartStore`.

### `useProductSpecs` (`src/hooks/useProductSpecs.ts`)
Derives displayable specs from a `ProductDetail` object:
- Filters out invalid values using `isValid()`.
- Maps specs to `{ icon, label, value }` tuples for rendering.
- Extracts and filters storage/color options.

## Zustand Stores

### `cartStore` (`src/store/cartStore.ts`)
- **State**: `cartCount: number`
- **Action**: `setCartCount(count)`
- **Persistence**: `localStorage` key `"cart-storage"`

### `detailStore` (`src/store/detailStore.ts`)
- **State**: `pageTitle: string`
- **Action**: `setPageTitle(title)`
- **Persistence**: `localStorage` key `"detail-storage"`
